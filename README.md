## **Days 1-30: Data Structures & Algorithms (DSA) Intensive**

### **Week 1 (Days 1-7): Core Data Structures**

- **Day 1:** **Arrays & Strings**
    - *Understand array manipulation and string algorithms.*
    - **Practice:** Solve array and string problems on [LeetCode](https://leetcode.com/).
    - **Resources:** [Cracking the Coding Interview](https://www.crackingthecodinginterview.com/)
- **Day 2:** **Linked Lists**
    - *Singly and doubly linked lists.*
    - **Implement:** Create linked list classes in Python.
    - **Practice:** Solve linked list problems.
- **Day 3:** **Stacks & Queues**
    - *LIFO and FIFO structures.*
    - **Implement:** Stack and queue classes.
    - **Practice:** Problems involving stacks and queues.
- **Day 4:** **Hash Tables**
    - *Hash functions and collision resolution.*
    - **Implement:** Simple hash map.
    - **Practice:** Use dictionaries to solve problems.
- **Day 5:** **Trees**
    - *Binary Trees and Binary Search Trees (BST).*
    - **Implement:** Tree node classes.
    - **Practice:** Tree traversal algorithms (in-order, pre-order, post-order).
- **Day 6:** **Heaps & Priority Queues**
    - *Understand min-heaps and max-heaps.*
    - **Implement:** Heap data structure.
    - **Practice:** Problems involving heaps.
- **Day 7:** **Review & Practice**
    - *Reinforce the week's concepts.*
    - **Activity:** Solve mixed data structure problems.
